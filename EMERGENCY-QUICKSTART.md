## 🔧 PRODUCTION UPGRADE (15 минут для реальных данных)

### **Опция A: Быстрая настройка Supabase**
1. **Создайте Supabase проект** (5 мин):
   - Идите на [supabase.com](https://supabase.com)
   - New Project → compliment-landing-mvp

2. **Добавьте API ключи** (2 мин):
   ```bash
   cp .env.example .env
   # Отредактируйте .env:
   VITE_SUPABASE_URL=https://your-project.supabase.co
   VITE_SUPABASE_ANON_KEY=your_anon_key_here
   VITE_MOCK_MODE=false
   ```

3. **Импортируйте схему** (3 мин):
   - SQL Editor → Вставьте код из `supabase-schema.sql`

4. **Перезапустите** (1 мин):
   ```bash
   npm run dev
   ```

**Детальная инструкция:** [SUPABASE-SETUP.md](SUPABASE-SETUP.md)
